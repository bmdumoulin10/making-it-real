HTML, CSS, JS

One thing I have been working on lately is learnign the basics to Web Development. Below is a table explaining some of the basics of HTML, CSS, and JS. Enjoy!

What Where How
HTML HTML is used to build the basic structure of your web page or website.
CSS CSS is used to add the style to your webpage or website and it corresponds to the HTML document you create.
JS JS, or Javascript, is used to build functions and add more functionality to your webpage or website. It also corresponds to the HTML document you create.

I hope you took the time to check out each of the links in order to get a better look into the beginning stages of web development.

Basic HTML Document Structure

Below you can check out the basic structure of an HTML document. Notice the top line, every HTML document must begin this way.

< !DOCTYPE html >

The above line is known as the declaration. by typing the html after !DOCTYPE you are letting the document know that this is going to be an HTML document.

Designing A Website for Profit

This beginner blog makes everything seem so easy. Like anything, if you take the time to learn the material, you too can design a website with your free time.   2. Make Pretty with CSS: In a seperate file, use your selectors to style your document so that it looks like you had imagined in your head.
